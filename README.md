This repo contains an approach to predicts the quality of Wines with following attributes of it,
  1. fixed acidity	
  2. volatile acidity	
  3. citric acid	
  4. residual sugar	
  5. chlorides	
  6. free sulfur dioxide	
  7. total sulfur dioxide	
  8. density	
  9. pH	
  10. sulphates	
  11. Alcohol
